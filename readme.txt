## Description

**Solar Calculation System** is a comprehensive WordPress plugin tailored for solar system installation companies. This plugin empowers businesses to efficiently manage and generate invoices while saving all relevant contract details in one place. With a user-friendly interface and a robust feature set, the Solar Calculation System streamlines the process of creating detailed solar panel and material calculations, converting estimates into invoices, and tracking payments (full or partial) against those invoices.

### Key Features:

- **User Login System:** Front-facing login for users to access the calculation tools.
- **Solar System Calculations:** Generate detailed calculations for solar panels, materials, wires, and more.
- **Invoice Generation:** Convert estimates into invoices with ease.
- **Payment Tracking:** Track full or partial payments against generated invoices.
- **Comprehensive Contract Management:** Store and manage all relevant details related to a contract within the plugin.
- **WordPress Integration:** Built entirely within WordPress, ensuring seamless integration with your existing setup.

## Installation

1. Upload the `solar-calculation-system` folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the **Solar Calculation System** settings page to configure the plugin to suit your needs.

## Usage

1. **User Login:** Users can log in through the front-end login system to access the calculation tools.
2. **Create Calculations:** Use the provided tools to generate detailed calculations for solar panels, materials, wires, and other components.
3. **Generate Invoices:** Convert your estimates into invoices with a single click.
4. **Track Payments:** Keep track of all payments, whether full or partial, associated with each invoice.
5. **Manage Contracts:** Store and review all contract details associated with your solar projects.

## Frequently Asked Questions

### Q: What versions of WordPress are supported?
A: The Solar Calculation System plugin is compatible with WordPress 5.0 and later.

### Q: Can I track partial payments with this plugin?
A: Yes, the plugin allows you to track both full and partial payments for each invoice.

### Q: Is the plugin compatible with other invoicing systems?
A: The Solar Calculation System is designed to be a standalone solution. However, compatibility with other systems may be added in future updates.
